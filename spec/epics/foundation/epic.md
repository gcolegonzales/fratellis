## Intent
Establish the buildable project, the locked visual language, and the shared page shell every other feature depends on. After this epic, `npm run build/test/lint` all pass on a styled site with working header/footer/navigation and base-path-safe links, and the swappable `submit()` seam exists.

## Features
- `FEAT-foundation-1` — Project scaffold & tooling
- `FEAT-foundation-2` — Design system implementation
- `FEAT-foundation-3` — Site shell (layout, header/nav, footer, base-path helper, submit seam)
